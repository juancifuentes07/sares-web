¿Cómo funciona?

Tres pasos para recuperar tu nota

El sistema combina tu historial de notas con un modelo de crecimiento matemático para calcular exactamente cuánto esfuerzo necesitas para pasar la materia.

01
📝
Regístrate
Crea tu perfil e inscribe las materias que estás cursando este semestre.
02
📊
Ingresa tu nota
Al finalizar el primer corte (30%), registra tu calificación en el sistema.
03
🔢
El modelo calcula
Runge-Kutta orden 4 proyecta tu curva de mejora y determina las iteraciones necesarias.
04
✅
Actúa
Recibes un número concreto de actividades extracurriculares para alcanzar el 3.0.
Características

Diseñado para el estudiante en riesgo

📈
Proyección con Runge-Kutta 4
El núcleo del sistema usa el método numérico RK4 sobre el modelo diferencial dy/dt = k·(5 − y), donde la nota crece hacia el máximo de 5.0 con una tasa ajustada según tu situación inicial. Cuanto más baja la nota, mayor es el coeficiente k y más actividades se calculan.
RK4
⚠️
Alerta temprana
Si tu nota proyectada no alcanza el 3.0 incluso con las actividades máximas, el sistema te alerta para que busques apoyo adicional a tiempo.
🎓
Por materia
Gestiona múltiples materias. Cada inscripción tiene su propio seguimiento independiente de notas y proyección.
🔗
API REST
Toda la lógica está expuesta como una API, lista para integrarse con otros sistemas académicos o aplicaciones móviles.
🧪
60 pruebas automatizadas
Validado con una suite completa de pruebas unitarias e integración usando pytest — el modelo matemático y los endpoints verificados.
