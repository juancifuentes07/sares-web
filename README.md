# SARES Web 
# Sistema de Alerta y Recomendación para Estudiantes en Riesgo Académico

## Tecnologías

- Python
- Flask
- Oracle Express Edition
- pytest
- REST API
- HTML / CSS / JavaScript

## Requisitos Previos

- Python 3.11
- Git
- Oracle Database Express Edition (XE)

## Instalación

1. Clonar el Repositorio
   - git clone https://github.com/juancifuentes07/sares-web
   - cd sares-web
2. Instalar dependencias
   - pip install -r requirements.txt
   - pip install cx_Oracle
3. Ejecutar el proyecto
   - python app.py
   - Servidor en: http://127.0.0.1:5000
